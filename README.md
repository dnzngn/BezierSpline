# Bezier Spline Editor

Unity Editor tool for creating dual-lane Bezier curves with interactive handles, node placement (Distance/Count modes), and mesh generation.

Runs in Edit Mode. Pure Bernstein polynomial math — no Vector3.Lerp.

## Setup

Open the project in Unity, then go to **Tools → Bezier Spline Editor**.

Developed and tested on **Apple M1 Silicon** with **Unity 6.3 LTS**.
